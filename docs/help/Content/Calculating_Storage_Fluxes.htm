In practice, integrals are approximately using the trapezium rule and time derivatives are approximated by the discrete difference of mean concentrations measured during consecutive time averaging intervals.

In eddy covariance applications where profile concentration data are not available, storage fluxes are approximated by using one-point time derivatives, assuming that all gradients nullify at ground level and that the profile is linear from the measurement point to the ground. Evidently, this is an extreme assumption that has little relation to the actual situation. This is why we encourage users to consider storage fluxes provided by EddyPro as purely indicative. For the same reason, storage fluxes are not summed to turbulent fluxes to provide Net Ecosystem Exchange (NEE) estimations, that would be affected by unacceptable inaccuracy.
