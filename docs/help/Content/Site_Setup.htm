Site Setup

Site information is entered in the GHG v7 (LI-7500A/LI-7200) software and recorded in the GHG files that are logged by the gas analyzer. To enter site information, install the PC software on your computer and connect with the gas analyzer.

The Site Setup window allows you to enter the site information that is required to compute flux results. This information is logged in a metadata file. You are not required to consider the metadata file explicitly, as it is created and modified in the gas analyzer configuration software and used unobtrusively by EddyPro and the SMARTFlux system. Nevertheless, all information is stored as plain text and can be retrieved and edited at any time.

The Site Setup window provides the following tabs:

  • USB Data Collection: See USB Data Collection.
  • Site Description: See Enter the Site Description.
  • Sonic Anemometer: See Enter Anemometer Information.
  • CO2/H2O Analyzer: See Enter CO2/H2O Analyzer Information.
  • CH4 Analyzer: See CH4 Analyzer Data.
  • Biomet: See Enter Biomet System Information.
